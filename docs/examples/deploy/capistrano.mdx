[Capistrano](https://capistranorb.com/) is a remote server automation and deployment tool written in Ruby.

## ðŸš€ Usage

Add the following command to your CI configuration file:

```bash
fluentci run --wasm capistrano deploy production
```

## Commands

| Name   | Description                       |
| ------ | --------------------------------- |
| setup  | Install capistrano.               |
| deploy | Deploy to a specific environment. |

## Code Examplesci

Add `fluentci-pdk` crate to your `Cargo.toml`:

```toml
[dependencies]
fluentci-pdk = "0.2.1"
```

Use the following code to call a module function:

```rust
use fluentci_pdk::dag;

// ...

dag().call("https://pkg.fluentci.io/capistrano@v0.1.0?wasm=1", "deploy", vec!["production"])?;
```

## CI/CD Integration

The following example shows how to integrate FluentCI with popular CI providers to set up Capistrano:

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";
import CodeBlock from "@theme/CodeBlock";

<Tabs>
  <TabItem value="gha" label="Github Actions">
    <CodeBlock language="yaml" title="ci.yml">
      {`name: 
on:
  push:
    branches:
      - main
jobs:
  tasks:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: Deploy
        uses: fluentci-io/setup-fluentci@v5
        with:
          wasm: true
          plugin: capistrano
          args: |
            deploy production 
      `}
    </CodeBlock>
  </TabItem>
  <TabItem value="gitlab" label="GitLab CI">
    <CodeBlock language="yaml" title=".gitlab-ci.yml">
