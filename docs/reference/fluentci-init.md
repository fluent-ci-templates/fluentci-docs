title: fluentci init
---

Initialize a new pipeline in the current directory.

```bash
fluentci init [pipeline] [options]
```

### Options

| Option | Description |
| ------ | ----------- |
| `--help`, `-h` | help for init |
| `--template`, `-t` | The name of the pipeline template to use. |
